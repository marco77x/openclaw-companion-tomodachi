# TOMODACHI Skill

Use this skill when the user wants to manage or interact with their TOMODACHI companion.

## Commands

```bash
~/.openclaw/scripts/companion/companion.sh hatch <user>
~/.openclaw/scripts/companion/companion.sh status <user>
~/.openclaw/scripts/companion/companion.sh talk <user>
~/.openclaw/scripts/companion/companion.sh feed <user>
~/.openclaw/scripts/companion/companion.sh play <user>
~/.openclaw/scripts/companion/companion.sh sleep <user>
~/.openclaw/scripts/companion/companion.sh groom <user>
~/.openclaw/scripts/companion/companion.sh cuddle <user>
~/.openclaw/scripts/companion/companion.sh train <user>
~/.openclaw/scripts/companion/companion.sh evening <user>
~/.openclaw/scripts/companion/companion.sh evoke <user>
~/.openclaw/scripts/companion/companion.sh decay
~/.openclaw/scripts/companion/companion.sh list
```
